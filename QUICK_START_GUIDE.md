# 🚀 RUNNINGHUB 功能面板 - 快速使用指南

## 🎯 **功能简介**

在HuanuCanvas画布的R按钮上方新增了🚀按钮，点击后在右侧出现矩阵图标形式的功能面板，每个图标对应一个RUNNINGHUB的WEBID功能。

## 📁 **配置文件位置**

**主要配置文件**: `src/data/settings.json`

## ⚡ **快速配置（3步完成）**

### **第1步: 打开配置文件**
```
打开文件: src/data/settings.json
```

### **第2步: 替换WEBID**
找到你需要的功能，将 `webappId` 从 `"your_webapp_id_here"` 替换为实际的RUNNINGHUB应用ID：

```json
{
  "id": "ai_image_gen",
  "name": "AI图片生成", 
  "icon": "🎨",
  "color": "#3B82F6",
  "webappId": "这里填入你的实际webappId",  ← 修改这里
  "category": "图片处理",
  "description": "基于提示词生成高质量图片",
  "defaultInputs": {}
}
```

### **第3步: 启动应用**
```bash
cd "D:\工作\Huanu\VibeCode\HuanuCanvas"
npm run dev
```

## 🎮 **如何使用**

1. **启动应用后进入Canvas工作区**
2. **点击画布左上角的🚀RUNNINGHUB按钮**
3. **在右侧面板中选择所需功能**
   - 支持搜索框快速查找
   - 支持分类筛选
4. **点击功能图标自动创建RunningHub节点**

## 🔧 **添加自定义功能**

在 `runningHubFunctions` 数组中添加新对象：

```json
{
  "id": "my_custom_function",
  "name": "我的自定义功能",
  "icon": "🆕",
  "color": "#FF6B6B",
  "webappId": "你的实际webappId",
  "category": "新分类",
  "description": "这是一个很棒的自定义功能",
  "defaultInputs": {}
}
```

## 🎯 **使用示例**

假设你想使用AI图片生成功能：

1. 在配置文件中找到 `"ai_image_gen"` 项
2. 将 `"webappId": "your_webapp_id_here"` 改为 `"webappId": "你的AI图片生成应用的实际ID"`
3. 保存文件，重启应用
4. 点击🚀按钮，选择🎨图标
5. 自动创建AI图片生成的RunningHub节点

## ⚠️ **注意事项**

- **修改配置后需要重启应用**
- **每个功能的webappId必须是你实际拥有的RUNNINGHUB应用ID**
- **图标使用Emoji格式，颜色使用HEX格式（如#3B82F6）**
